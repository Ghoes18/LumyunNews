
# LumyunNews

A website of articles and news made for my final course project.


## Run Locally

Clone the project

```bash
  git clone https://github.com/Ghoes18/LumyunNews
```

Go to the project directory

```bash
  cd LumyunNews
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```


## Tech Stack

**Client:** EJS, HTML5, CSS3

**Server:** Node, Express
